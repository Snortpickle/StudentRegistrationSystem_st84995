# DESIGN.md

## Purpose

This file defines the visual and structural design contract for the Student Registration System UI.

The goal is to guide AI-assisted frontend generation so that the interface remains consistent, accessible, and connected to the existing MVVM backend logic.

## Framework Choice

Framework: WPF XAML

Reason:

The existing application is a WPF MVVM desktop application. The UI should be implemented in XAML and connected to ViewModel commands instead of creating a separate web interface.

## Feature Scope

Selected feature:

Reset Student and Course Filters

The UI must provide clear controls that allow the user to reset active filters and return to the full student or course list.
